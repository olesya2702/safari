Safari для macOS:

  • Впервые Safari был включён в состав Mac OS X 10.3 Panther.
  • Начиная с Mac OS X 10.4 Tiger, Safari является единственным браузером, который поставляется в комплекте с ОС.
  • В версии Mac OS X 10.6 Snow Leopard браузер Safari 4 стал 64-разрядным (как и большинство других системных приложений)[4].

11 июня 2007 года Apple выпустила тестовую версию Safari 3.0 для Windows XP и Vista.

По утверждению Майкла Чейшпергера, Safari для Windows загружает и отображает страницы быстрее, чем Internet Explorer 7 Mozilla Firefox однако отрыв на самом деле не настолько велик, как это представляется Apple, и связано это с тем, что производительный тест iBench, разработанный компанией VeriTest, даёт преимущество Safari перед другими браузерами.
